Scoring — формулы

  • Ассистент — PTS + AST*3 + REB*1.3 + BLK*3 + STL*3 + TO*(-2)
  • Ребаундер — PTS + AST*1.5 + REB*2.6 + BLK*3 + STL*3 + TO*(-2)
  • Чернорабочий — PTS + AST*1.5 + REB*1.3 + BLK*6 + STL*6 + TO*(-2)
  • Всё включено — базовая формула; если в каждом показателе ≥1 → +7
  • Сюрприз — базовая формула; если итог ≥20 → +10
